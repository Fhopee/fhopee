## 第 1 页

We providing an independent control system Level 2 (working station) of whole auto packaging line, if any additional information from front line Level 3 (SCADA)
needs to be connected to our system Level 2, It needs additional software development.
There are mainly below 3 methods for information input in working station for the coil feeding at the turnstile.
1. Manually details input on working station.
2. Scanning labels to obtain information.
3. Adopt a Industrial Ethernet method called “ProFINet” to transfer your PLC’s information to our PLC system.
The information required:
A. Mother coil size in OD x ID x Width
B. Strip sort order
C. Stacking pattern: for example, width 20x5 pcs per pallet. (it would be setting by manually)
