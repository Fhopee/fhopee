## 如何生成完整 Markdown 文件

在项目根目录执行：

- **命令行**：`pip install pypdf python-docx` 后运行 `python extract_to_rag.py`
- **或双击**：`run_extract.bat`

生成后，本目录下会出现与上表对应的 `.md` 文件，每个文件包含可被 AI/RAG 直接读取的纯文本内容。
