# 新加坡线-验机后-问题点

新加坡线：

1.屏显参数列表：

屏显中的“参数列表”，需由目前的9行增加至20行；

2.列表数量：

目前一个列表对应着4个十字臂；需增加3个列表分别对应4个臂。现场的包装堆垛需要根据客户的要求调整，所以程序除了满足提供的复杂堆垛要求，也要能胜任普通的包装堆垛要求。

3.备件表：

表1：详细备件列表，供随机发货用；

表2：推荐一份备件列表，供客户额外采购备用！

4.包装工位: （添加 开On / 关 Off 按钮）  

增加一个 开On / 关 Off 按钮, 供客户现场选择启用或者不启用缠绕包装功能。

5.堆垛工位： （添加“确认 OK”按钮）

增加一个木条“确认”按钮，供人工放置木条后确认用。

6.整条线负载：

需整理一份表格：电机分布及对应功率大小，并核算整线总功率。

补漆及机器表面处理工作：

发货前，需做好整条线机器表面补漆细节，保证机器外观。

备喷漆罐：

发货前，需备喷漆罐，随机器一起发货，供客户现场补漆用。
