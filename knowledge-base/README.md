# Knowledge Base / 知识库

此目录用于存放项目的参考文档、SEO 策略、内容素材等 Markdown (.md) 文件。

## 如何使用
1. 将你的 `.md` 格式的策略文档、关键词列表、竞品分析等文件放入此目录。
2. 在与 AI 助手对话时，使用 `@文件名` (例如 `@seo-strategy.md`) 来引用这些文档，让 AI 基于这些资料进行创作。

## 建议的文件
- `seo-strategy.md`: 包含关键词优化、内外链策略等。
- `brand-guidelines.md`: 品牌语调、写作规范。
- `product-knowledge.md`: 补充的产品技术细节。
